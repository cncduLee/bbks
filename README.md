bbks
====

E-reading platform(电子阅读WEB端)		

##简介		

bbks是基于图书搜索的电子阅读、社交平台。系统分为：图书搜索系统，社交互动系统以及电子阅读系统。		
其中图书搜索系统分为：数据抓取系统、数据集成系统和图书检索平台。该应用采用Appache开源协议。

##	技术	

###	backend	
			
*	SpringMVC freamwork
* 	SpringData
* 	Hibernate3.x
*	Lunce+CK
*	Appache shiro
* 	encache		


###	frontend
*	Jquery and plugins
*	bootstrap
*	css3
